# FoodFetch - Fast & Fresh Food Delivery

A modern, high-performance food ordering platform built with Next.js, React, and GenAI.

## Features

- **AI Recommendations**: Personalized food and restaurant suggestions powered by Google Gemini.
- **AI Image Generation**: Dynamic, high-quality food photography generated on-the-fly using Genkit and Imagen/Gemini models.
- **Smart Search**: Find your favorite dishes or cuisines with instant filtering.
- **Responsive UI**: Professional, mobile-first design using ShadCN UI and Tailwind CSS.
- **Cart & Checkout**: Seamless ordering experience with multiple payment options (Card, Cash on Delivery).
- **Dark Mode**: Fully supported dark and light themes for late-night cravings.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Components**: ShadCN UI (Radix UI)
- **AI/ML**: Genkit + Google Gemini 2.5
- **Icons**: Lucide React

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Set up your `.env` with `GEMINI_API_KEY`.
4. Run the development server: `npm run dev`.
